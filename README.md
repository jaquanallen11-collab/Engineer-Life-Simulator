## Architecture

### Metal 4 Pipeline
```swift
- MetalRenderer.swift: Main rendering engine
- Metal4MLPipeline.swift: ML inference pipeline  
- Shaders.metal: Particle rendering
- AdvancedShaders.metal: Post-processing effects
```

### SwiftUI Views
```swift
- ModernViews.swift: Start, Exam, University views
- ModernGameViews.swift: Main game, scenarios, predictions
- ModernShopViews.swift: Shop and portfolio interfaces
```

### Game Logic
```swift
- GameModels.swift: Data structures
- MLPredictor.swift: CoreML integration
- GameStore: State management
```
